Pixeloids

by @nicolasleao

Procedural monster avatars for static sites, scripts, prototypes, and anywhere else a seed should always resolve to the exact same creature.

Background Toggle the container backdrop on or off without changing the creature itself.
Idle animation Seeded micro-motions like bobbing, blinking, looking around, and tiny sways.

Examples

Use anywhere

Generate deterministic monster SVG strings in static pages, scripts, SSR, or any app that can render a string.

Animated Toggle all code snippets between static and animated usage.

Animated: off

React / Next.js — Static 
import { createSvg } from 'pixeloids';

export default function Avatar({ seed }) {
  const svg = createSvg(seed, {
    size: 128,
    background: true
  });

  return <div dangerouslySetInnerHTML={{ __html: svg }} />;
}
React / Next.js — Animated 
import PixeloidsAnimations from 'pixeloids/animations.mjs';

export default function AnimatedAvatar({ seed }) {
  const svg = PixeloidsAnimations.createAnimatedSvg(seed, {
    size: 128,
    variant: 'monster',
    background: true
  });

  return <div dangerouslySetInnerHTML={{ __html: svg }} />;
}
