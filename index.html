Zero dependency · deterministic · inline SVG

Pixeloids

Tiny framework-agnostic avatar generator for sharp, blocky monster characters. Same seed, same creature.

Examples

Use anywhere

The API stays DOM-free: generate deterministic monster SVG strings in static pages, scripts, SSR, or any app that can render a string. 

<script src="./pixeloids.js"></script>
<script>
  const svg = Pixeloids.createSvg('demo-seed', {
    size: 160,
    background: true
  });

  document.querySelector('#slot').innerHTML = svg;
</script>
const Pixeloids = require('./pixeloids.js');

const avatar = Pixeloids.createAvatar('ops-bot', {
  size: 128,
  background: true
});

console.log(avatar.svg);
console.log(avatar.metadata);

API: createSvg · getMetadata · createAvatar · toDataUri
