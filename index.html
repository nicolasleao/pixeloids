Pixeloids

by @nicolasleao

Procedural monster avatars for static sites, scripts, prototypes, and anywhere else a seed should always resolve to the exact same creature.

Background Toggle the container backdrop on or off without changing the creature itself.

Examples

Use anywhere

Generate deterministic monster SVG strings in static pages, scripts, SSR, or any app that can render a string.

Browser 
<script src="./pixeloids.js"></script>
<script>
  const svg = Pixeloids.createSvg('demo-seed', {
    size: 160,
    background: true
  });

  document.querySelector('#slot').innerHTML = svg;
</script>
Node / CommonJS 
const Pixeloids = require('./pixeloids.js');

const avatar = Pixeloids.createAvatar('ops-bot', {
  size: 128,
  background: true
});

console.log(avatar.svg);
console.log(avatar.metadata);

API: createSvg · getMetadata · createAvatar · toDataUri
